OTA Solution

This project aims to use OTA (Over the Air) technology in a simple way to flash an application code on an ECU (Electronic Control Unit) without the need of direct wired connectivity.

Why use OTA ?!
  • Let's say that we have an ECU in a closed system and it needs an update all the products with the same system are required to install the update, the process we'll go through to update the products one by one is not practical at all so using OTA w could update all the products at the same time provided that they have an internet connection.
  • The OTA technology is used in the automotive industry to update ECUs in cars, This method saves much time and money than calling in the vehicles for a software update.
